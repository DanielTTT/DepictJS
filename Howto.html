Learn how to use Depict.js

Depict.DefineObject(object,properties)

Depict.DefineObject takes two aruguments.



var Rectangle=Depict.DefinObject("RectangleObject",{

    x:10,
    y:10,
    width:100,
    height:100,
    fill:"blue",
    transprancy:1,
  })

var Circle=Depict.DefinObject("CircleObject",{

    x:10,
    y:10,
    width:100,
    height:100,
    fill:"blue",
    transprancy:1,
  })

Property argument has to be defined as a object. It takes 5 parameters.
Now objects are created.


object.draw()

When the draw() method is called, the object will appear.


Rectangle.draw();
Circle.draw();

object.migrate(x,y,speed,function)

object.migrate() is a method for moving objects.You can define the coordinates,speed and a function executes after the migration is compleated. (You don't have to define the function)


Rectangle.migrate(300,300,1000,function(){
  alert("Migrated")
});
Circle.migrate(400,400,1000,function(){
  alert("Migrated")
});

object.fadeIn(speed)
object.fadeOut(speed)

When you use object.fadeIn(),you dont have to call the object.draw() method.


Rectangle.fadeIn(1000);
Circle.fadeOut(1000);
